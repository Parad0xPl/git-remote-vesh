# Git Enc
Simple tool for remote SSH repository inside VeraCrypt container

## Dependency
### Windows
* [SSHFS-Win](https://github.com/winfsp/sshfs-win)
* [VeraCrypt](https://www.veracrypt.fr/en/Home.html)
### Linux
* [sshfs](https://github.com/libfuse/sshfs)
* [VeraCrypt](https://www.veracrypt.fr/en/Home.html)
